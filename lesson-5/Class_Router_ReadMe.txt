Класс Router содержит 2 метода: init и resources.
Метод init вызывает метод resources, регистрирующий контроллер и доступные для него методы (привязка маршрута к конкретным именам метода). После у статического метода PostsController вызывается функция connection.  
Класс PostsController содержит только сами методы-обработчики. Чтобы он мог обрабатывать события (переданный маршрут), при помощи директивы Extend модуль Resource, который вызывает функцию по переданному маршруту.
